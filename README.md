# Wonderkid Identification in European Football

This project investigates whether performance metrics from a U-21 player's first top-5 European league season can predict the probability that they become a regular contributor (≥1500 minutes per season) at a top-5 club by age 24. It compares a gradient boosting classifier against simple heuristic baselines, with explicit calibration analysis and temporal validation to avoid data leakage. Built as a portfolio project during my Engineering Physics MSc as part of my transition towards Data Science.
