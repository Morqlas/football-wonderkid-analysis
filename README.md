# Wonderkid Identification in European Football

This project investigates whether it is possible to predict if an U-21 football player outside the top-5 European Leagues is likely to make a move to top-5 leagues by the age of 24, using only publicly available data. It compares a gradient boosting classifier against simple heuristic baselines, with explicit calibration analysis and temporal validation to avoid data leakage. Built as a portfolio project during my Engineering Physics MSc as part of my transition towards Data Science.
